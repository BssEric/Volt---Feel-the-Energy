⚡ VOLT — Feel the Energy

VOLT é um projeto de front-end desenvolvido com foco em experiência visual imersiva, inspirado em sites premiados (Awwwards). A proposta é apresentar uma marca fictícia de bebidas energéticas com forte apelo visual, interações fluidas e design moderno.

🚀 Tecnologias utilizadas
HTML5
CSS3
JavaScript
Bootstrap
GSAP (animações avançadas)
WebGL (efeitos visuais)
🎯 Objetivo do projeto

O objetivo foi criar uma landing page com:

Alto impacto visual
Animações suaves e envolventes
Interatividade com o usuário
Estrutura moderna de front-end
Design inspirado em produtos reais

💡 Funcionalidades
🎞️ Hero section com carrossel interativo
🎨 Mudança dinâmica de cores conforme o produto
🥤 Animação da lata (rotação e transições)
📱 Layout responsivo
⚡ Scroll com animações (GSAP)
🧭 Navbar minimalista e moderna
🧠 Conceito criativo

A ideia central foi transmitir energia, movimento e intensidade — elementos diretamente conectados ao universo de bebidas energéticas.

Cada produto possui sua própria identidade visual (cor, sensação e ritmo), criando uma experiência única durante a navegação.

Durante o desenvolvimento, foram explorados conceitos como:

Animações avançadas com GSAP
Organização de layout responsivo
Performance em sites visuais
Design focado em storytelling
